# MyRead
This is a React project / A Book Lending App

## Getting Started

### Prerequisites

- Nodejs(npm)

### Installing

```bash
$ git clone https://github.com/miri4ech/pj-myreads.git
$ cd pj-myreads
```

```bash
$ npm install 
```

## Running the project

```bash 
$ npm start
```

You can see the site locally. [http://localhost:3000/](http://localhost:3000/)
