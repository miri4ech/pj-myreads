# MyRead

## Requirements

- Nodejs(npm)

## Installation

```bash
$ git clone https://github.com/miri4ech/pj-myreads.git
$ cd pj-myreads
```

```bash
$ npm install 
$ npm update
$ npm install serve -g
```

## Build

```bash 
$ serve -s build 
```

You can see the site locally. [http://localhost:3000/](http://localhost:3000/)
